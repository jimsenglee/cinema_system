# Frontend Folder

This folder is reserved for the React frontend application.

**To add your frontend:**
1. Copy your React app files into this folder
2. Make sure you have `package.json`, `src/`, `public/` folders
3. Run `npm install` to install dependencies
4. Run `npm start` to test the frontend
